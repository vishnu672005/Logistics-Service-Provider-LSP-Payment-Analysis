# Logistics-Service-Provider-LSP-Payment-Analysis
Project Title / Headline  
A dynamic and interactive Power BI dashboard designed to monitor logistics invoice processing and payment workflows—focusing on ageing analysis, payment delays, vendor prioritization (MSME vs Non-MSME), and operational efficiency to improve financial control and decision-making.

Short Description / Purpose  
This LSP Payment Analysis Dashboard is an interactive Power BI report that helps analyze invoice volume, ageing, payment priorities, vendor classification, and processing efficiency. It is designed to support finance teams in tracking delays, prioritizing MSME payments, and improving overall invoice workflow management.

Tech Stack  
The dashboard was built using the following tools and technologies:  
• 📊 Power BI Desktop – Used to create interactive visuals and dashboard layout  
• 📂 Power Query – Used for cleaning, transforming, and preparing invoice data  
• 🧠 DAX (Data Analysis Expressions) – Used to calculate KPIs like total invoices, total amount, and delayed counts  
• 🧩 Data Modeling – Created calculated columns for ageing, priority, and vendor type  
• 🎨 UI/UX Design (Power BI) – Designed clean layout with slicers and visual hierarchy  
• 📁 File Formats – .pbix for development and .png for dashboard preview  

Features / Highlights  

• Business Problem :  
Logistics finance teams handle a large volume of invoices daily, making it difficult to track payment status and ensure timely processing. Key questions like:

How many invoices are delayed?  
What is the total outstanding payment?  
Which invoices need immediate attention?  
How much payment is pending for MSME vendors?  
Which employees are handling the workload?  

…are not easy to answer using raw data.

• Goal of the Dashboard :  
To build an interactive dashboard that:  
Provides a clear overview of invoice and payment status,Tracks ageing and identifies delayed invoices,Highlights MSME vendors requiring priority handling,Analyzes employee workload distribution,Enables quick and data-driven financial decisions  

• Walkthrough of Key Visuals  

Key KPIs (Top Section) :  Total Invoices ,Total Payable Amount,Delayed Invoice Count .These provide a quick snapshot of overall performance.

Priority Filter (Slicer) : Allows filtering by Delayed, First Priority, Second Priority, MSME Priority .Helps focus on critical invoices instantly  

Vendor Type Filter (Slicer) :  MSME vs Non-MSME . Helps track compliance-related payments  

Ageing Distribution (Bar Chart) :  Shows invoice amounts across ageing buckets (0–7, 8–30, 31–60, Not Due)  .Helps identify overdue payment zones  

Amount by Priority (Bar Chart) : Displays payment exposure across priority levels  . Helps understand urgency of payments  

Vendor Type Analysis (Donut Chart) :  Shows MSME vs Non-MSME payment share  .Useful for compliance tracking  

Top Vendors (Bar Chart) :  Displays vendors with highest outstanding amounts.Helps identify major contributors  

Holder Performance (Bar Chart) :  Shows number of invoices handled by each employee  .Helps analyze workload distribution  

• Business Impact & Insights  
Operational Efficiency: Identifies delays and improves invoice processing  
Financial Control: Tracks outstanding payments and reduces risk exposure  
MSME Compliance: Ensures priority handling of MSME vendors  
Workload Management: Helps distribute work efficiently among employees  
Data-Driven Decisions: Enables faster and smarter financial decisions  

Key Findings / Results  
The analysis shows that a significant portion of payment value lies in delayed invoices, indicating processing inefficiencies. MSME vendors require strict priority handling due to compliance requirements. Certain vendors contribute heavily to outstanding payments, increasing financial exposure.  

Holder-wise analysis reveals uneven workload distribution, where a few employees manage most invoices. Ageing distribution shows many invoices falling into higher ageing buckets, highlighting delays in approval and processing.  

Overall, the dashboard provides actionable insights to improve turnaround time, optimize workload, and enhance payment efficiency.
